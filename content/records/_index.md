这里记录 AI agent 在本机上执行过的任务，包括关键决策、操作过程和反思。

每条记录包含：

- **主体归因** — 哪个 agent 做的（codebuddy / explorer / investigate 等）
- **结果评估** — 成功 / 部分完成 / 失败 / 经验学习
- **状态追溯** — 操作前后的上下文快照
- **代码锚点** — 关联的 git commit，可直接回滚
- **经验沉淀** — 反思与教训

按日期归档，支持多 agent 区分。
